## Ключевые метрики

```
CFE rate (без YACCA) : 20 / 288 = 6.9% bit-flip'ов приводят к CFE
CFE rate (с YACCA)   : 21 / 640 = 3.3%

Из 20 CFE (незащищённая программа):
  YACCA обнаружил    :  7  (35.0%)  ← detection rate
  YACCA сделал BENIGN: 12           (YACCA подавил CFE без ошибки)
  YACCA пропустил    :  0           (silent CFE — необнаруженные)
```
